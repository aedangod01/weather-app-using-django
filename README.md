# Weather App Using Django

A simple weather application built with Django.  
This project fetches and displays current weather information for a given city.

## Features

- Search weather by city name
- Display temperature and weather condition
- Clean and simple UI
- Built with Django framework

## Technologies Used

- Python
- Django
- HTML / CSS
- Weather API
